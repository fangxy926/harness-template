# 登录页面 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 创建一个自包含的静态登录页面 `login.html`，使用纯 HTML + CSS + JS，支持表单验证、密码可视性切换、登录成功模拟。

**Architecture:** 单文件设计，CSS 内联在 `<style>` 块，JS 内联在 `<script>` 块，零依赖，无构建工具，直接浏览器打开运行。

**Tech Stack:** HTML5, CSS3（Flexbox、CSS 变量）, 原生 JavaScript（ES6+）

### Task 3: JavaScript 交互逻辑

**Files:**
- Modify: `login.html`（替换 `<script>` 块内容）

- [ ] **Step 1: 将以下 JS 替换掉 Task 1 中的 `/* Task 3 将填充此处 */` 占位注释**

```javascript
(function () {
  'use strict';

  const form = document.getElementById('login-form');
  const usernameInput = document.getElementById('username');
  const passwordInput = document.getElementById('password');
  const toggleBtn = document.getElementById('toggle-pwd');
  const errorMsg = document.getElementById('error-msg');
  const submitBtn = document.getElementById('submit-btn');

  // ── 错误提示工具 ──────────────────────────────────────────
  function showError(msg) {
    errorMsg.textContent = msg;
  }

  function clearError() {
    errorMsg.textContent = '';
  }

  // 输入框 focus 时清除错误
  usernameInput.addEventListener('focus', clearError);
  passwordInput.addEventListener('focus', clearError);

  // ── 密码可视性切换 ────────────────────────────────────────
  toggleBtn.addEventListener('click', function () {
    const isPassword = passwordInput.type === 'password';
    passwordInput.type = isPassword ? 'text' : 'password';
    toggleBtn.textContent = isPassword ? '🙈' : '👁';
  });

  // ── 表单验证 ──────────────────────────────────────────────
  function validate() {
    const username = usernameInput.value.trim();
    const password = passwordInput.value;

    if (!username) {
      showError('请输入用户名');
      usernameInput.focus();
      return false;
    }
    if (!password) {
      showError('请输入密码');
      passwordInput.focus();
      return false;
    }
    if (password.length < 6) {
      showError('密码至少 6 位');
      passwordInput.focus();
      return false;
    }
    return true;
  }

  // ── 表单提交 ──────────────────────────────────────────────
  form.addEventListener('submit', function (e) {
    e.preventDefault();
    clearError();

    try {
      if (!validate()) return;

      // 模拟 loading 状态
      submitBtn.disabled = true;
      submitBtn.textContent = '登录中…';

      setTimeout(function () {
        submitBtn.disabled = false;
        submitBtn.textContent = '登录';

        // 展示成功提示
        clearError();
        const successEl = document.createElement('p');
        successEl.className = 'success-msg';
        successEl.textContent = '登录成功 ✓';
        // 若已存在成功提示则替换
        const existing = form.querySelector('.success-msg');
        if (existing) existing.remove();
        form.appendChild(successEl);
      }, 500);
    } catch (err) {
      showError('发生未知错误，请重试');
      submitBtn.disabled = false;
      submitBtn.textContent = '登录';
    }
  });
}());
```

- [ ] **Step 2: 浏览器验证交互行为**

  逐项检查：
  1. 空表单提交 → 显示「请输入用户名」
  2. 填用户名不填密码 → 显示「请输入密码」
  3. 密码填 `abc`（3位）→ 显示「密码至少 6 位」
  4. 用户名填 `test`，密码填 `123456` → 按钮变「登录中…」→ 500ms 后显示「登录成功 ✓」（绿色）
  5. 点击 👁 → 密码变明文，图标变 🙈；再点 → 恢复密码模式，图标变 👁
  6. 输入框点击 → 错误提示消失
  7. 浏览器控制台无报错
